# City Sandbox · 上帝视角沙盒建造

基于 **Kenney Retro Urban Kit**（低多边形 3D 城市套件）的上帝视角沙盒建造游戏。
地图、道路、建筑、树木、载具全部来自同一套 3D 资源 —— **地图的真实度与其它物体完全一致**。

## 特性
- **自动生成地图**：按种子程序化生成完整城市（道路网 + 人行道 + 高低错落的建筑群 + 公园森林 + 街道设施 + 载具）。点 `🎲 重新生成` 或输入种子复现。
- **无限资源（创造模式）**：随意放置，没有任何消耗。
- **上帝视角**：可平移 / 旋转 / 缩放的斜俯视相机。
- **建造系统**：8 类调色板（建筑预制件、地面/道路、树木、墙体构件、道具、载具、地形、删除），共 120+ 个模型。
- **存档**：保存 / 读取到浏览器本地存储。

## 运行
需要本地 HTTP 服务器（ES 模块 + GLB 加载不能用 `file://` 直接打开）。任选其一：

```bash
# 方式 A：Node（仓库内置的小型静态服务器）
node server.mjs
# 然后浏览器打开 http://localhost:8772

# 方式 B：Python
python3 -m http.server 8772
# 然后打开 http://localhost:8772
```

## 操作
| 操作 | 按键 |
|---|---|
| 放置 / 涂刷（可按住拖动） | 鼠标左键 |
| 旋转视角 | 右键拖动 |
| 平移视角 | 中键拖动 / 空格 + 左键拖 / `WASD` |
| 缩放 | 滚轮 |
| 转向 | `Q` / `E` |
| 旋转待放置的物体 | `R` |
| 切换删除（橡皮擦） | `X` |
| 取消选择 | `Esc` |

橡皮擦：先删除格子上的物体；若该格没有物体，则把地面重置为草地。

## 资源说明
- 使用的是 `kenney_retro-urban-kit`（GLB 3D 模型）。引擎与加载器已本地化到 `assets/vendor/`，**无需联网**。
- `kenney_retro-textures-fantasy` 是 **2D 中世纪贴图**（非 3D 模型），与低多边形 3D 城市风格不统一，为保证"同一真实度"未混入本作。它更适合做一个独立的 2D 俯视模式 / 奇幻主题（可后续扩展）。
- 资源版权：Kenney（CC0），见 `assets/KENNEY-*-License.txt`。

## 代码结构
```
index.html        UI 外壳 + importmap
server.mjs        本地静态服务器
src/
  main.js         启动、预加载、顶栏、渲染循环
  engine.js       场景/渲染器/光照/上帝相机/GLB 模型库
  world.js        网格数据、地面/物体分层、程序化地图生成、存档
  buildings.js    由墙体方块堆叠而成的预制建筑
  ui.js           调色板、ghost 预览、放置/涂刷/删除交互
  noise.js        种子随机数 + 值噪声
  manifest.js     自动生成的模型清单（分类 / 尺寸 / 预览图）
assets/
  models/         124 个 GLB + Textures/
  previews/       调色板缩略图
  vendor/         本地化的 three.js + GLTFLoader
```
